刚接触vue的前端路由,和后端的controller有点类似,但是不需要刷新界面的机制又让我想起来了ajax,ajax是动态渲染,而前端路由可以随便拿数据展示.个人感觉前端路由很强大,所以这方面的知识必须掌握,特此记录.

这里面设计一些es6语法的导入导出,这里就不细讲了,说一下export default,就是不指定对象名将对象导出,

让导入者自定义对象名.

这里记录一个简单的demo,自己详解一下里面的具体内容以及流程

1)、main.js

import Vue from 'vue'
import App from './App.vue'
import router from './router/index'

Vue.config.productionTip = false / /阻止启动生产消息，常用作指令

new Vue({
  render: h => h(App),    //es6箭头函数语法 h是参数,而h(App)是返回值,这里的render会直接渲染还回组件
  router
}).$mount('#app')  //$mount('#app')等价于el: '#app',事实上后者会被vue源码加载渲染变成前者.

程序入口,这里有个最重要的东西vue实例.

查了一些语法,一些人故意炫技将东西说的天花乱坠,本人看了一点recat基础,这边的render语法其实是一种简写:

等价于

import App from './App.vue'
new Vue({  
​            el: "#app",
​            components: {
​                "App": App  
​            }
​        });
并自动将<App></App>加载在<div id="app"> </div>
    即:
<div id ="app">
    <App></App>
</div>

2)、App.vue

<template>
  <div id="app">
    <!--replace相当于HTML5的history.replaceState(),并不会入栈操作,直接渲染,浏览器没有返回箭头-->
    <router-link to='/home' tag="button" replace>首页</router-link>
    <router-link to='/about' tag="button" replace>关于</router-link>
    <br/>
    --------------------------------------------------------------
    <br/>
    <button @click="homeClick">首页</button>
    <button @click="aboutClick">关于</button>
    <router-view></router-view> <!--占位显示路由-->
  </div>
</template>

<script>

export default {
  name: 'App',
  components: {

  },
  methods: {
    homeClick(){
      //通过代码修改路由   $router是属于router插件内部的
      //this.$router.push('/home'); //入栈方式
      this.$router.replace('/home');//替换方式
    },
    aboutClick(){
      this.$router.replace('/about');
    }
  },
}
</script>

<style>
</style>

App.vue可以理解为主组件,界面由多个组件组成,render函数会直接渲染并加载主组件.

3)、router/index.js

//配置路由相关的信息
import VueRouter from 'vue-router'  //导入vue插件  导入之前安装
import Vue from 'vue'       //导入vue  vue来安装vue-router

//以下为需要渲染的组件
import Home from '../components/Home.vue'
import About from '../components/About.vue'

//1.vue的use安装这个插件
Vue.use(VueRouter);

//2.创建routes对象参数
const routes = [
    {
        path: '/',
        redirect: '/home'

    },
    {
        path: '/home',
        component: Home
    },
    {
        path: '/about',
        component: About
    }
]
//导出对象并设置为HTML5的history模式
export default new VueRouter({
    routes,
    mode: 'history'
})

路由配置文件,设置路由映射规则以及让vue加载这个组件.
