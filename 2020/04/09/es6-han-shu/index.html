ES6笔记之函数

ES6和ES5的区别：

  1. 变量
  2. 函数
  3. 数组
  4. 字符串
  5. 面向对象
  6. Promise
  7. generator
  8. 模块化

函数

ES6：箭头函数

  1. 如果只有一个参数，（）可以省

  2. 如果只有一条语句，{}可以省

  3. 函数的参数: 参数扩展/展开

  4. 默认参数

    //通常函数的定义方法
var fn1 = function(a, b) {
  return a + b
}
//使用ES6箭头函数语法定义函数，将原函数的“function”关键字和函数名都删掉，并使用“=>”连接参数列表和函数体。
var fn1 = (a, b) => {
  return a + b
}

// 无参
var fn1 = () => {}

// 单个参数
  var fn2 = function(a) {}
  var fn2 = a => {}

// 多个参数
var fn3 = function(a, b) {}
var fn3 = (a, b) => {}

参数扩展

  1. 收集剩余参数

    function show(a,b,...args){
console.log(args); //[8,1,2,5,3,4];
}
show(5,6,8,1,2,5,3,4);

    参数扩展必须写在最后一个参数的位置 …args

  2. 扩展运算符… 展开数组

    let arr1 = [1,2,3];
let arr2 = [4,5,6];

let arr3 = [...arr1,...arr2];
console.log(arr3); //[1,2,3,4,5,6]

    展开后的效果，跟直接把数组内容写在这一样的

  3. 默认参数

    //ES6可以这样设置默认参数值
function show(a,b = 5,c = 12){
console.log(a,b,c); 
};
show(99); //99,5,12
show(5,2,1); // 5,2,1
